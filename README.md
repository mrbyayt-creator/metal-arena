# METAL ARENA — safe bot spawn fix

Built from the previous corner unstuck AI build.

Changes in this package are limited to bot spawn placement:

- Offline campaign/survival bots no longer spawn on the outer edge ring near containers/walls.
- Bot spawn points now require extra clearance from walls/containers and other bots.
- Spawn fallback points were moved to safer open areas.

Other gameplay systems were not intentionally changed.
