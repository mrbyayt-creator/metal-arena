# METAL ARENA — Bigger pickups visual fix

Base: V5 follow-up build with no jump and kept machine-gun aim assist.

Only pickup visuals were changed:

- health kits are larger and brighter;
- ammo pickups have larger visible bullet models;
- turbo/nitro pickups are larger and more expressive;
- all pickups have a stronger colored glow and base ring.

Pickup logic, balance, respawn, collision radius, weapons, AI, networking and other gameplay were not intentionally changed.


Patch: added team-mode left HUD only: compact red/blue score and CS-style kill feed for TDM/round lobbies.
