# METAL ARENA — decor and bot rocket fix

Node.js WebSocket server + browser client.

## Local run

```bash
npm start
```

Open:

```text
http://localhost:8080
```

## Railway

Deploy from GitHub. Railway uses `package.json` and `railway.json`.

Start command:

```bash
npm start
```

## Update notes

This build keeps the previous V4 polish base and only adjusts:

- bot rocket visual/contact explosion against the player;
- varied procedural box/container textures;
- extra low-poly arena decor with collision: lamps, crates, barrels, tires, scrap fences.
