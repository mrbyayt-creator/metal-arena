# METAL ARENA — V4 weapon rework fix

Built from the previous V4 decor/rocket build.

Changes in this package are limited to the requested weapon behavior fixes:

- Mortar splash uses full 3D distance and a slightly larger area.
- Mines have proximity trigger radius and explode immediately when a vehicle drives near them.
- Heavy cannon projectile explodes on vehicle hit and has very slight homing/aim help.
- Tesla now creates a balanced blue radial shockwave around the car.
- Railgun now fires an instant blue beam instead of a flying ball projectile.

Other gameplay systems were not intentionally changed.


## Weapon tuning update

- Heavy cannon homing reduced to a very small correction.
- Heavy cannon area/splash damage removed; it still shows an impact explosion on direct hit.
- Railgun beam hit width and visual beam thickness increased slightly.
- Projectile/vehicle hit shell reduced so it wraps closer to the car body with only a small safety margin.
